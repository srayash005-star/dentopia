# Dentopia

Making a website for the dental clinic of my mom.

A starter project for Dentopia.

## How to open

1. Open VS Code.
2. Go to `File` > `Open Folder...`
3. Select `c:\Users\sraya\.vscode\dentopia`.

## Recommended setup

- Install Node.js (already done)
- Install VS Code extensions: ESLint, Prettier, Volar/Vetur, Angular Language Service

## Run locally

```bash
npm install
npm start
```
